**响应格式**
```
{
    "code": 1,
    "data": [],
    "msg": "操作成功！"
}
```
<br>
<br>

**状态码说明**

| 状态码（code）   | 含义说明（msg）                                        |
| -------------- | ----------------------------------------------------- |
| 1              | 调用成功                                               |
| 0              | 调用失败，系统异常或其他原因                              |
